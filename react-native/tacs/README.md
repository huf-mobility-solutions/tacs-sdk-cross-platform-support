# react-native-tacs

React Native Wrapper for the Liberkee TACS SDK

## Installation

```sh
npm install react-native-tacs
```

## Usage

```js
import Tacs from "react-native-tacs";

// ...

const result = await Tacs.multiply(3, 7);
```
